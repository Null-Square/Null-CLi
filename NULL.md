# NULL.md

This project is a fork of Qwen Code (itself adapted from Gemini CLI), being rebranded and extended as "Null CLI" for penetration testing research.

During the transition, most functionality and developer workflows remain the same as described in `QWEN.md`. Please refer to that document for detailed development conventions until this file is fully expanded for Null.

Key differences planned for Null:

- CLI name and bin: `null`
- Package scope: `@null/…`
- Security tooling focus (e.g., Nmap, Gobuster, SQLMap) with methodology alignment (MITRE ATT&CK)
- Real-time logs and reporting for offensive security workflows
