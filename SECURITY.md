# Security Policy

Use Null CLI only against systems you own or have explicit permission to test.

Please report security issues privately to the NullSquare maintainers. Do not include secrets, customer data, exploit payloads against third-party systems, or non-public run traces in public issues.

This repository intentionally excludes NullSquare managed-platform internals. Pull requests that add non-public orchestration, external callback infrastructure, customer artifacts, or credential attack automation will be rejected.
