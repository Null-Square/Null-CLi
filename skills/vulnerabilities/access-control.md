name: Basic access-control review
description: Shallow public guidance for access-control observations.
category: vulnerability
---

Record access-control observations only when the operator supplied authorization and test accounts.

Evidence floor:

- The tested endpoint or object.
- The expected authorization boundary.
- The observed response showing missing or weak enforcement.

This public skill does not include private replay automation or advanced IDOR validation logic.
